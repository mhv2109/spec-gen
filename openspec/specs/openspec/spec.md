### Requirement: Validatespecstructure

The system SHALL validates the overall structure of an openspec document, including headings, sections, and markdown formatting.

#### Scenario: ValidSpecStructure
- **GIVEN** A properly formatted OpenSpec document
- **WHEN** validateSpecStructure is called
- **THEN** Returns a ValidationResult with valid: true and no errors or warnings

#### Scenario: MissingTitle
- **GIVEN** An OpenSpec document without a title (# heading)
- **WHEN** validateSpecStructure is called
- **THEN** Returns a ValidationResult with valid: false and an error about missing title
